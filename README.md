## 📊 Project Overview

This project leverages **Artificial Intelligence and Machine Learning techniques** to provide a comprehensive analysis of Netflix's userbase using Power BI. The dashboard utilizes **intelligent data processing** and **predictive analytics** to visualize key metrics including:

| Category | Metrics | AI Application |
|----------|---------|----------------|
| 📈 **Revenue** | Monthly revenue trends by quarter and year | **Predictive forecasting** for future revenue |
| 🌍 **Users** | Global user distribution across countries | **Pattern recognition** in user behavior |
| 💳 **Subscriptions** | Basic, Standard, Premium plan analysis | **Intelligent segmentation** for targeted campaigns |
| 📱 **Devices** | Smartphone, Smart TV, Laptop, Tablet usage | **Anomaly detection** in device usage patterns |
| 👥 **Demographics** | User distribution by age group | **AI-driven demographic analysis** |
